# FUTURE_CS_02 - Phishing Email Detection & Awareness System

## Objective
Analyze phishing and legitimate email samples to identify phishing indicators and improve user awareness.

## Tools Used
- Gmail simulation
- Manual inspection
- Browser tools
- MS Word / PDF

## Steps Performed
1. Created a phishing email sample
2. Created a safe email sample
3. Compared sender domains, message tone, and links
4. Identified phishing indicators
5. Classified email risk levels
6. Documented findings and prevention tips

## Findings
- Fake sender domain detected
- Urgent wording used to create panic
- Suspicious link identified
- Legitimate email used official domain and secure link

## Risk Level
- Phishing Email: High
- Safe Email: Low

## Prevention Tips
- Verify sender email domain
- Avoid clicking unknown links
- Check for spelling mistakes in domains
- Use MFA
- Do not trust urgent messages without verification
